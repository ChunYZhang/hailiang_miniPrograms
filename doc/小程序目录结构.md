# 小程序端目录结构

## 根目录结构

```
miniPrograms/
├── .swc/                     # SWC 编译器缓存
├── config/                  # 项目配置
├── dist/                    # 编译输出目录（微信小程序代码）
├── node_modules/            # 依赖包
├── src/                     # 源代码（使用 Taro 框架）
├── .swc/                    # SWC 编译器插件
├── babel.config.js          # Babel 配置
├── package.json             # 项目依赖配置
├── taro.config.ts           # Taro 构建配置
├── tsconfig.json            # TypeScript 配置
└── project.config.json      # 微信小程序项目配置
```

## 源代码目录 (src/)

```
src/
├── app.config.ts            # 小程序全局配置（页面路由、窗口样式等）
├── app.tsx                  # 小程序根组件
├── components/              # 公共组件
│   ├── CartModal/           # 购物车弹窗组件
│   │   ├── index.scss
│   │   └── index.tsx
│   └── PindanBar/           # 拼单栏组件
│       ├── index.scss
│       └── index.tsx
├── config/                  # 配置文件目录
├── pages/                   # 页面目录
├── services/                # 服务层（API 接口）
│   └── api.ts               # API 接口封装
├── store/                   # 状态管理
│   └── index.tsx            # 状态管理入口
└── styles/                  # 全局样式
    ├── index.scss           # 全局样式
    └── index.ts             # 样式配置
```

## 编译输出目录 (dist/)

编译后生成的微信小程序代码，包含编译后的 `.js`、`.wxml`、`.wxss`、`.json` 文件。

### 页面结构 (dist/pages/)

每个页面编译后包含：
- `index.js` - 页面逻辑
- `index.wxml` - 页面结构
- `index.wxss` - 页面样式
- `index.json` - 页面配置

## 技术栈

- **框架**: Taro（支持跨平台小程序开发，基于 React）
- **语言**: TypeScript
- **状态管理**: React Hooks + Context
- **样式**: SCSS
- **构建工具**: SWC
